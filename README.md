# Todo list app

### Features
* [x] Add a task
* [x] Remove a task (swipe it to the left)
* [x] Preserve state on device
* [x] Biometrics (expo local-authentication)
* [x] Dark mode
* [x] Scalable architecture
* [x] zustand store
* [x] expo
* [x] typescript
* [x] unit tests

### How to start
```bash
npm run start
```
